# Lightweight astromaterials

Collection of lightweight lunar rocks, based NASA Astromaterials data with simplified mesh and compressed textures. The goal is to have a collection of lightweight 3D lunar rocks to aid lunar surface simulation environments run faster.

Of course, if simulation time or compute is not a constriant, it would be better to use original 3D models in full resolution for visual fidelity.

## Credits
